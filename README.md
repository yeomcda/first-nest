## Description

nestjs 프레임 워크를 이용한 첫번째 프로젝트
- module, service, controller 에 대한 이해
- module 의 분리 (movies)
- entities(model) 생성
- dto 를 통한 validation
- unit test, e2e test

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
